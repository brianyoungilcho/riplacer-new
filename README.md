# riplacer-new

## Getting started

Prereqs: Node.js + npm (install via nvm if needed).

```sh
# Install dependencies
npm i

# Start the dev server
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Use Vercel or your preferred platform. If using Vercel CLI:

```sh
npx vercel
```
